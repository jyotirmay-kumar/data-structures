# Data-Structures
Project to learn &amp; practice data structures in Java

___


### Linked List
* Interface - ``LinkedList<T>``
* Implementations - ``SinglyLinkedList<T>``, ``DoublyLinkedList<T>``
____

### Stack
* Interface - ``Stack<T>``
* Implementations - ``ArrayStack<T>``, ``LinkedStack<T>``

----

### Queue
* Interface - ``Queue<T>``
* Implementations - ``ArrayQueue<T>``, ``LinkedQueue<T>``

----

### Tree
* BinaryTree - ``BinaryTree<E>``
